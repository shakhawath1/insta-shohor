JavaScript is a high-level interpreted programming language. It runs inside an Internet browser, read the file line-by-line top to bottom. Like a waterfall. Browser use V8 engine for execute a JavaScript file. When a web page load in a browser, first the HTML and CSS file are load. After the loading browser create DOM for execute the JavaScript file. By creating the DOM being updated by JavaScript code and rendere by the browser. The order here is important. If the JavaScript did not wait for the HTML and CSS loading to finish, it would not be able to change the DOM elements.

An event loop is a process that waits for the call stack to be cleared before returning calls from the task stack to the call stack. When the stack is cleared, the event node is activated. Check the task queue for available callback calls. If there is, it pushes the call stack, waits for the call stack to be freed again, and repeats the same process.
